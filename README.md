**Student Grade Manager**

**Project Description**: This project involves building a simple GUI based application to manage student grades. The system will let users add, view, update, and delete student records containing name, ID, and marks for three subjects. The system will also calculate the average and determine grade (A–F).

**Features:**
  •	Add a new student record
  •	Display all student records
  •	Update marks of a specific student
  •	Delete a student record
  •	Calculate average and assign grade
  
**Additional Functionalities:**
  •	Input Validation: Validates names, IDs, and numeric ranges
  •	Replay Option: User can continue or exit
  •	Scalable Design: More subjects or grading rules can be added easily
